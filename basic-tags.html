This text is in the em elements of html. This means that the text is italicized, as you can see. This is text in the i element of html. This has the same effects as the em where it italices the letters but instead this element is used when writing in different languages or using special terms. This text is obviously bolded. You can get bolded text by using the strong element in html. The strong element is used to highlight that the text is important. Similarly this text is bolded as well.
