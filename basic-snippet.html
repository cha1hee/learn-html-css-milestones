Disecting the Snippet

<p>
  Want to search the web?
  Try <a href="https://google.com" id="the-best-link" class="banana"  target="_blank">Google</a>!
</p>

<p>

This is the starting tag for a paragraph text. It automatically fills the entire line of the webpage which can be changed through CSS styling.

</p>

This is the ending tag for a paragraph text. It indicates the end of the paragraph. This tag is necessary for every time there is a starting paragraph tag, or there will be an error in your code.

<a></a>

This is the tag used to input a link into a webpage. In the above snippet it is shown as a child tag of the paragraph tag. You can tell that it is a child from the indentation of the lines which show hierarchy and it is within the opening and closing paragraph tag.

Attributes Within the Link Tag

href="https://google.com"

The href attribute is the URL that tells the browser where to go. It holds the web address to redirect the user to the target link.

id="the-best-link"

The id attribute is used to specify the object to a name that is unique to that object. It is important to note that you cannot have more than one element with the same id.

class="banana"

The class attribute is used to specify an overarching type name to an object/element. It is used to group different objects to a class name.

target="_blank"

The target attribute in a link tag specifies where the linked document is being opened to.
